# zbashfile — 示例 Shell 环境（安全模式）

本目录是项目提供的**示例 Shell 环境**，专用于 Termux **故障安全模式（Failsafe）**。该目录中的 `.bashrc` 配置了精简的学科工具命令（数学、化学、语文等），在安全模式下 `source` 后即可使用，让你在修复环境中也能调用项目中的常用学科功能。

> 本目录也包含一批独立的 Bash 脚本（倒计时、数学计算、二次函数、数列、字符串处理、文件查看等），可作为功能参考或临时使用。所有 `.sh` 脚本都可以用 `bash 文件名.sh` 来运行。部分脚本支持在命令后加参数（如 `bash mypi.sh 1000`）。

## ⏱ 倒计时与时间工具

### djs.sh — 高精度倒计时器
**精确到百分之一秒的倒计时器。**

📖 **新手使用**：
```bash
bash djs.sh
# 输入倒计时长度（单位：秒），例如：
# 300 → 倒计时 300 秒（5 分钟）
# 100 → 倒计时 100 秒（1 分 40 秒）
# 75  → 倒计时 75 秒（1 分 15 秒）
```
从输入的数值开始递减到 0，每秒刷新显示剩余时间，到达后输出 `time out!`。

✨ **特点**：利用 `EPOCHREALTIME` 实现毫秒级精度；支持任意小数秒输入；进度实时刷新。

### math.sh — 数学计算启动器
**根据参数选择不同的数学运算模式。**

📖 **新手使用**：
```bash

```
启动后进入交互模式，在 `>>>` 后输入表达式，输入 `quit` 退出。

✨ **特点**：统一入口调度器；启动时自动显示当月日历；`-l` 参数切换运算模式。

**纯整数运算的交互式计算器，支持变量赋值。**

✨ **特点**：轻量无依赖；支持变量存储中间结果（`x=5`）；50次操作后自动退出防沉迷。

## 🛟 故障安全模式（Failsafe）

本目录专为 Termux **故障安全模式**设计。当 Termux 无法正常启动（例如 `.bashrc` 语法错误、环境变量错乱等），可进入安全模式后 `source` 本目录的 `.bashrc`，获得一个精简的学科工具环境。

### 使用方法

在 Android **桌面长按 Termux 图标** → 点击 **"故障安全"**（中文版）或 **"Failsafe"**（英文版），进入安全模式后执行：

```bash
source zbashfile/.bashrc
```

之后即可使用配置好的学科命令（数学、化学、语文等），但**只能使用系统命令**。

### 适用场景

| 场景 | 说明 |
|------|------|
| **启动闪退** | .bashrc 或其他配置文件语法错误 |
| **环境变量错乱** | PATH 被改乱导致所有命令找不到 |
| **包管理器卡死** | apt/pkg 损坏无法正常修复 |
| **需要快速访问学科工具** | 安全模式下临时使用数学/化学/语文命令 |
