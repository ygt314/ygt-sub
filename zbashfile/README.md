# zbashfile 目录文件列表

该目录包含Termux环境下使用的Bash脚本和工具文件。

## 文件列表

### 日期相关
- [date\_for.sh](./date_for.sh) - 日期处理脚本

### 数学计算
- [deg.sh](./deg.sh) - 角度计算脚本
- [math.bc](./math.bc) - bc数学计算器脚本
- [math.sh](./math.sh) - 数学计算脚本
- [math\_f.sh](./math_f.sh) - 浮点数学计算
- [math\_s.sh](./math_s.sh) - 简单数学计算
- [mypi.sh](./mypi.sh) - 圆周率计算
- [sqrt.sh](./sqrt.sh) - 平方根计算

### 字符串处理
- [str\_float.sh](./str_float.sh) - 浮点数字符串处理
- [str\_int.sh](./str_int.sh) - 整数字符串处理
- [str\_num.sh](./str_num.sh) - 数字字符串处理

### 实用工具
- [djs.sh](./djs.sh) - JavaScript相关工具
- [ds10.sh](./ds10.sh) - 十进制处理
- [dss.sh](./dss.sh) - 字符串处理工具
- [fabc\_ahk.sh](./fabc_ahk.sh) - AHK脚本相关
- [fbdsh.sh](./fbdsh.sh) - 后台处理脚本
- [fblq.sh](./fblq.sh) - 文件处理工具
- [hello.exe](./hello.exe) - 示例程序
- [jst.sh](./jst.sh) - JavaScript测试
- [nbran.sh](./nbran.sh) - 分支处理
- [pabc\_ahk.sh](./pabc_ahk.sh) - AHK处理脚本
- [pf.sh](./pf.sh) - 文件处理
- [pff.sh](./pff.sh) - 文件过滤
- [pi.awk](./pi.awk) - AWK圆周率计算
- [rtime.sh](./rtime.sh) - 时间处理
- [sd333.sh](./sd333.sh) - 特殊数据处理
- [setter.sh](./setter.sh) - 设置工具
- [terset.sh](./terset.sh) - 终端设置

- [yabcr.sh](./yabcr.sh) - YABC运行工具

## 使用说明

这些脚本主要用于Termux环境下执行各种数学计算、字符串处理和 系统操作任务。

### 运行方法

```bash
bash script_name.sh [参数]
```
