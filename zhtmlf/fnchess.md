# 函数棋 (FnChess)

## 【游戏简介】

函数棋是一款融合数学函数与策略对战的益智游戏。玩家通过构造数学函数表达式，使其图像穿过目标格子来得分，同时需要避开禁止区。表达式越简短，得分越高。

## 游戏规则

### 【游戏模式】
- **本地对战**：两位玩家在同一设备上轮流对弈
- **人机对战**：挑战 AI 对手 Summa
- **闯关模式**：81+ 道预设关卡，无时间限制，逐关解锁
- **测试模式**：自由输入函数验证图像


### 【棋盘与坐标】
棋盘使用标准数学坐标系，向上为正 Y，向右为正 X。
每个格子 (x, y) 代表 x ∈ [x, x+1]、y ∈ [y, y+1] 的区域。

棋盘随回合数自动扩大：
- 第  1— 4 回合 ±5 范围（10×10）
- 第  5— 8 回合 ±6 范围（12×12）
- 第  9—12 回合 ±7 范围（14×14）
- 第 13—16 回合 ±8 范围（16×16）
- 第 17—20 回合 ±9 范围（18×18）
- 第 21—24 回合 ±10 范围（20×20）

注：闯关模式棋盘固定为 20×20，不随回合扩展。


### 【每回合流程】
每个回合依次经历六个阶段：

  1. 选目标  — 对手选择若干绿色目标格（函数图像需穿过）
  2. 设禁止区 — 对手设置若干红色禁止格（函数图像不可进入）
  3. 锁元素  — 对手锁定某些数学元素，本回合本方不可使用
  4. 输入函数 — 本方从元素栏构造函数表达式，计时器倒计时
  5. 系统判定 — 自动绘制图像，检测命中与违规
  6. 计分切换 — 得分结算后，轮换到另一玩家

本地对战：第一回合由玩家 B 设目标、玩家 A 构造函数；此后奇偶回合轮流互换。


### 【难度与目标格数量】
- **简单 (Easy)**：每回合 1 个目标格
- **普通 (Normal)**：每回合 2 个目标格
- **专家 (Expert)**：每回合 3 个目标格


### 【时间限制】（超时扣 1 分；闯关/测试模式无限制）
- 第 1—4 回合   40 秒
- 第 5—8 回合   50 秒
- 第 9—12 回合  60 秒
- 第 13—16 回合 70 秒
- 第 17—20 回合 80 秒
- 第 21—24 回合 90 秒


### 【禁止区规则】
- 前 4 回合没有禁止区
- 第 5 回合起出现禁止区，数量随回合增加：
    - 第 1—8 回合  最多 1 个
    - 第 9—16 回合 最多 2 个
    - 第 17—24 回合 最多 3 个


### 【元素锁定规则】
- 前 4 回合不允许锁定
- 第 5—12 回合最多锁 1 个
- 第 13 回合起最多锁 2 个
- 同一元素每局最多被锁 2 次
- x、(、) 不可被锁定；简单难度下四则运算符也不可被锁定


### 【得分规则】
- 完美命中（穿过所有目标格且未进禁止区）  →  1—5 分（按简洁度）
- 误入禁止区                              →  −1 分
- 未命中所有目标格                        →  −1 分
- 超时未提交                              →  −1 分

所有回合结束后，总分高者获胜；相同则平局。


### 【简洁度评分】
表达式的**token（词元）**越少，得分越高：
- 1—2 个 token  →  5 分（极简大师）
- 3—5 个 token  →  4 分（简洁高手）
- 6—9 个 token  →  3 分（常规表达）
- 10—15 个 token → 2 分（稍微复杂）
- 16 个以上      →  1 分（冗长表达）

Token 计算：每个独立的变量(x)、数字、常量(π/e/i)、运算符(+ − × ÷ ^ !)、
函数名(sin/cos/tan/abs/ln/sqrt) 各计 1 个 token，括号不计。

> 示例：<br>
  x         → 1 token  → 5 分<br>
  12.34  → 1 tokens → 5 分<br>
  sin(x)    → 2 tokens → 5 分<br>
  2x+1      → 4 tokens → 4 分（2、x、+、1，其中隐式乘法不计）<br>
  x^2+2x+1  → 8 tokens → 3 分<br>


### 【支持的数学元素】
- **变量**：x
- **常量**：0—9、π、e、i（虚数单位，i²=−1）
- **运算符**：+  −  ×  ÷  ^（乘方）  !（阶乘）
- **函数**：sin  cos  tan  abs（绝对值）  ln（自然对数）  sqrt（平方根）
- **括号**：(  )

支持的函数类型：所有的函数


### 【闯关模式关卡分布】
- 第 1—29 关   简单（熟悉基础函数）
- 第 30—53 关  中等（函数组合技巧）
- 第 54—69 关  困难（高阶复合函数）
- 第 70—81 关  专家（???）
- ???           神秘难度（??????）

**通关条件**：命中所有目标格且未进入禁止区。可无限重试，进度自动保存。


### 【常见问题】

**Q**：函数没命中是什么原因？
**A**：①图像只是擦过格子边缘（需真正穿过内部，详见附录）；②函数没有穿过所有的目标格，或穿过了禁止区；

**Q**：超时了怎么办？
**A**：超时自动扣 1 分，下一回合继续。

**Q**：历史格子是什么？
**A**：之前回合用过的目标格和禁止区会以半透明灰色显示，不可再次选择，仅供参考。


## 附录：图像穿过格子判定算法：

系统判断函数图像是否穿过某个格子，采用"视觉采样 + 自适应容差"方式：

- 第一步：折线段采样
系统将函数曲线看作一系列折线段，在每段折线上均匀取 21 个采样点
（含两端），所有采样点共同覆盖整条曲线的走势。

- 第二步：包围盒快速剔除
对每一段折线，先检查其包围盒是否与目标格有重叠。若不重叠，
直接跳过该段，大幅加速检测。这一步过滤掉 95% 以上的无关线段。

- 第三步：自适应容差判定
对每个采样点，系统要求它必须落在格子内部才算命中，仅擦边界不算。
具体规则：格子四边各向内缩进一个"容差"作为安全边距，采样点必须
落入缩圈后的内部区域。

**容差**的计算方式：
  基础容差 = 0.8 像素（以屏幕像素为单位）
  容差 = 0.8 × 5 ÷ 当前坐标系范围

  例如：
- 范围 ±5（早期回合）→ 容差 0.80 像素
- 范围 ±8（中期回合）→ 容差 0.50 像素
- 范围 ±10（后期回合）→ 容差 0.40 像素

范围越大、容差越小，意味着后期判定比早期略宽松。
这一点容差非常微小，肉眼几乎无法察觉，正常穿过格子的图像不会受影响。
只有当曲线仅仅"将将擦过"格子边缘时，才可能被容差过滤掉。

- 第四步：命中判定
任意折线段上的任意采样点满足"落入格子内部（含容差缩减后）"
即判定为命中该格。所有采样点均不满足则判定为未命中。

因此，构造图像时请确保函数曲线肉眼可见地穿过了目标格内部，
尽量避免紧贴格子边缘的刁钻走法。
有时，即使图像数学意义上穿过了格子内部，也会被判定为擦过格子边缘。
